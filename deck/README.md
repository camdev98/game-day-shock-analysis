# Deck
 
`MIS_3116_Final_Presentation.pdf` — the 15-minute final presentation deck delivered for MIS 3116 (Data Visualization), fall 2025 at Iowa State.
 
## Structure
 
The deck walks the audience through the project in the same order as the analysis itself:
 
1. **Setup** — what Ames looks like on a home football Saturday and why "every game is a windfall" is worth testing.
2. **Methodology** — datasets, filters, the home-game schedule frame, and the comparison axes (Control / Regular Game / Cy-Hawk; Early / Night kickoff).
3. **Preparation findings (liquor)** — the dataset reframing from consumer to B2B, then the day-of-week breakdown showing Cy-Hawk Thursday as the only meaningful spike.
4. **Crash findings** — volume comparisons, severity distributions, and the impaired-driver share showing 13× higher rate on night games.
5. **Conclusion** — three takeaways (liquor only spikes for Cy-Hawk; crash volume scales with population but per-capita risk holds; night games are the impaired-driver outlier) and recommendations for Ames businesses, ISU operations, and city services.
## Note on authorship
 
This was a five-person team deliverable. I built the liquor section (slides covering the dataset reframing, the day-of-week breakdown, and the related recommendation). The crash section was built by a teammate; both are included so the synthesis at the end of the deck makes sense.
